создание svg картинки (круг, радиус 50px, координаты 50 по X и 50 по Y, цвет оранжевый) .svg { задаем свойства контейнеру svg } -- прямоугольник x=20px y=50px - расположение верхнего левого угла фигуры rx="20" ry="50" -- скругление краев по x и по y -- многоугольник с координатами вершин (5-по x, 135 - по y) -- эллипс, rx - радиус по x, ry - радиус по y -- построение линии, ширина 5px цвет - violent stroke="red" и stroke-width="5" -- обводка, можно использовать со всеми фигурами, половина обводки залезает на фигуру, половина снаружи -- ломанная линия, отличаеться от poligon тем что обводка сама по себе не замыкаеться fill-opacity= "0.75" -- прозрачность фона svg stroke-opacity = "0.5" -- прозрачность обводки stroke-linecap = "butt"/"round"/"square" -- обводка на концах линий, по умолчанию/скругление на концах(становить длинее)/доп обводка на концах stroke-linejoin = "milter/round/bevel" -- обводка на углах, по умолчанию/угол закругляеться/тупой угол(складываеться как лента) stroke-dasharray="50 10" -- делает линию пунктиром, 50 - длина отрузков, 10 - длина пробелов stroke-dashoffset="25" -- сдвиг пунктира на 25 (можно исп отрицательное значение)
